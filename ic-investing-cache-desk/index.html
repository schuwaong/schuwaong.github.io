WHAT THE AGENTS DO

Plain English

What each agent does.

Each agent does one job, so you can scan faster, avoid obvious mistakes, and compare ideas the same way.

Setup Finder

Finds stocks near an entry, add-on, or pullback zone so traders can start with a shorter watchlist.

Trend Check

Shows whether momentum and price action are helping or fading, so you do not chase weak moves.

Value Check

Compares price with valuation context so investors can avoid overpaying.

News Watch

Flags headlines, policy, and macro shifts so you can explain sudden moves before you size up.

Risk Challenge

Looks for the weakest part of the idea before you act, so shaky setups get challenged early.
