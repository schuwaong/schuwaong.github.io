# Instagram Edits Workflow

Use this as the practical first version until the product has approved Meta publishing access.

## Jarvis Generates

For each listing, Jarvis creates:

- 8-15 second vertical MP4.
- Optional voiceover audio.
- Square or carousel image post.
- Caption, hashtags, and CTA.
- A short editor checklist.

## User Finishes In Instagram Edits

1. Open Instagram Edits on the phone.
2. Import the generated MP4.
3. Add trending audio, stickers, or extra text animation if needed.
4. Check the first 1 second hook.
5. Export or share to Instagram.
6. Paste the generated caption.

## Why This Version

This keeps posting manual and avoids password-based browser automation. The app can still feel automated because the hard part is done: listing intake, creative angle, video draft, voiceover, caption, and hashtags.

## Later Version

When a customer wants true scheduling, use Meta's official Instagram publishing API with user authorization, not stored Instagram passwords.
