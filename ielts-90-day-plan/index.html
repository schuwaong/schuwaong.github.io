How to use it / 使用方法

Simple enough to keep using every day.

The plan uses 3 phases and 15 six-day cycles. Every cycle repeats the same learning rhythm: spot the task, learn words, copy a model, produce your own answer, do one timed set, then review and repeat.

Step 1 / 第 1 步 Read the question first, then Chinese.

Start with the English prompt, check the translation, and then say the task back in simple words.

Step 2 / 第 2 步 Build a useful word log.

Write the word, the meaning, the example, and the Chinese note. This is how weak language becomes repeatable marks.

Step 3 / 第 3 步 Use short answers first.

If you are stuck, write the simplest correct English. Short and clear beats a blank page.

Step 4 / 第 4 步 Review every sixth day.

The plan is built around a repeatable six-day loop so you can revise, test, and recover without guessing what to do next.

Study rule / 学习规则

Learn one thing, practice it once, mark it once, and review it once. That small loop is enough to change the way a revision day feels.

90-day route / 90天路线

Three phases. Fifteen cycles. Ninety days.

Phase 1 builds the base, Phase 2 grows skill, and Phase 3 locks in exam mode. Keep the same routine, and let the task type change instead of the method.

0 / 90 done

Phase 1 / 第 1 阶段

Build the base / 打基础

Days 1-30 are for learning the test format, the question types, and how to spot answers faster.

Goal / 目标 Know what each IELTS section wants before you chase speed. 先看懂雅思每个部分要什么，再去追速度。

Phase 2 / 第 2 阶段

Build skill / 提升能力

Days 31-60 move from notes to answer patterns: speak more smoothly, write cleaner paragraphs, and use better examples.

Goal / 目标 Turn vocabulary into natural answers and answers into marks. 把词汇变成自然的答案，把答案变成分数。

Phase 3 / 第 3 阶段

Exam mode / 考试模式

Days 61-90 are for timed full tests, weak-skill repair, and calm review before the real exam day.

Goal / 目标 Stay calm, check the question type, and finish strong. 保持冷静，先看题型，稳稳做完。

Vocabulary bank / 小词汇库

Words, translations, and example lines.

Search a word, then read the translation and the example sentence. This is the fastest way to build usable IELTS language.

Sentence frames / 句型框架

Ready-made lines for answers.

Use these when you do not know how to start. They help with speaking, Task 1, Task 2, and short written answers.

Chinese learner tips / 中文学习者技巧

Small habits that make the page easier to use.

These are the little moves that keep Chinese-speaking students from getting stuck when the English gets heavy.
