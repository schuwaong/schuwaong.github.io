How Becky uses it / Becky 怎么用

One card at a time.

This page is intentionally simple: choose a card, finish the task, tick Done, and the card disappears. The page keeps the words, frames, and Chinese support nearby.

1 / 第 1 步 Pick one card.

Do not plan the whole day first. Start with the next visible card.

2 / 第 2 步 Use Chinese to understand.

Read the Chinese note quickly, then answer in English.

3 / 第 3 步 Tick it when finished.

Completed cards hide automatically, so the page feels lighter.

4 / 第 4 步 Learn one harder word.

The daily popup is now for stronger IELTS vocabulary and examples.

Becky's rule / Becky 的规则

One visible card is enough to start. The goal is steady practice, not a perfect mood.

Checklist / 勾选清单

Finish a card. Watch it disappear.

Done cards are hidden by default so Becky only sees what still needs attention. Turn on Show done to review or untick something.

Phase 1 / 第 1 阶段

Build the base / 打基础

Days 1-30 are for learning the test format, the question types, and how to spot answers faster.

Goal / 目标 Know what each IELTS section wants before you chase speed. 先看懂雅思每个部分要什么，再去追速度。

Phase 2 / 第 2 阶段

Build skill / 提升能力

Days 31-60 move from notes to answer patterns: speak more smoothly, write cleaner paragraphs, and use better examples.

Goal / 目标 Turn vocabulary into natural answers and answers into marks. 把词汇变成自然的答案，把答案变成分数。

Phase 3 / 第 3 阶段

Exam mode / 考试模式

Days 61-90 are for timed full tests, weak-skill repair, and calm review before the real exam day.

Goal / 目标 Stay calm, check the question type, and finish strong. 保持冷静，先看题型，稳稳做完。
