NLP Keyboard

Rewrite messages before you send them.

NLP Rephrase Keyboard is an Android keyboard prototype that helps draft clearer, softer, more useful replies for dating, sales, support, negotiation, and everyday chats.

View Android project

Modes

  • Empathize
  • Clarify
  • Soften
  • Ask a better question
  • Close politely

Privacy Model

The recommended setup is phone to local Mac server to logged-in local AI tooling. The keyboard should draft text for review, not auto-send messages.

Local Endpoint

Use a reachable endpoint such as http://YOUR_MAC_IP:8000/rephrase or a temporary HTTPS tunnel when away from the same Wi-Fi.
